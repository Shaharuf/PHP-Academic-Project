# Inventory Management System
This is a web-based inventory management system built using PHP and MySQL. It allows users to manage their inventory, track their products, and generate reports.

# Features
1. Secure user authentication system with different user roles (admin and employee)
2. Dashboard showing an overview of inventory statistics
3. Ability to add, edit, and delete products
4. Ability to categorize products
5. Ability to track product quantities and status (in stock, low stock, out of stock)
6. Ability to generate sales and inventory reports
7. User-friendly interface

# Requirements
1. PHP version 7.0 or higher
2. MySQL version 5.6 or higher
3. Apache web server
4. Web browser (Chrome, Firefox, Safari, etc.)

# Installation
1. Clone the repository or download the ZIP file.
2. Extract the files to your web server's directory.
3. Create a new MySQL database and import the inventory.sql file included in the project.
4. Modify the database connection settings in the config.php file to match your database credentials.
5. Access the application by navigating to the URL where the files were extracted.

# Usage
1. Login using the credentials provided or create a new account with the appropriate role.
2. Navigate to the dashboard to view an overview of inventory statistics.
3. Add products using the "Add Product" button and fill out the form.
4. Edit or delete products using the buttons on the product listing page.
5. Generate reports using the appropriate buttons on the dashboard.
